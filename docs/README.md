# Plobie Technical Documentation

This directory contains technical documentation for the Plobie project.

## Directory Structure

- `architecture/` - ERDs, sequence diagrams, technical decisions
- `setup/` - Development environment setup guides

## Note on Sensitive Files

**IMPORTANT:** The following subdirectories are .gitignored and should NEVER be committed:
- `budget/` - Cost estimates, hour tracking
- `contracts/` - Client contracts, proposals
- `proposals/` - SOWs, project scope documents

Any files with these keywords are also automatically excluded: budget*.xlsx, cost*.xlsx, hours*.xlsx, contract*.pdf, proposal*.pdf, scope*.xlsx

Keep all PM and financial documents in these folders or your local Drive/Dropbox only.
